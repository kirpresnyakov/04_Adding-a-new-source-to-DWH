Описание проекта
Цель: интеграция нового источника данных (external_source) в DWH и построение инкрементальной витрины для анализа активности заказчиков.

Реализованные задачи: Изучение нового источника – анализ структуры и содержания таблиц craft_products_orders и customers в схеме external_source.

Загрузка данных в DWH – скрипт loading_data_into_dwh.sql переносит данные из нового источника в схему dwh.

Создание витрины dwh.customer_report_datamart – DDL-скрипт create_datamart.sql определяет структуру витрины с полями:

идентификатор и ФИО заказчика
потраченная сумма и доход платформы (10%)
количество заказов, средний чек, медианное время выполнения
самая популярная категория и самый популярный мастер
статусная аналитика по заказам (созданы, в процессе, в доставке, завершены, незавершены)
отчётный период (год и месяц)

Инкрементальный расчёт витрины – скрипт обновления использует логику, аналогичную витрине мастеров, с применением оконных функций.

Результат: новый источник интегрирован, витрина готова к использованию.

Файлы проекта:

src/scripts/check_source_data.sql – проверка данных в новом источнике

src/scripts/loading_data_into_dwh.sql – загрузка данных в DWH

src/scripts/create_datamart.sql – создание витрины customer_report_datamart
